## ğŸ“ Learning Paths

### The 3-Phase System

Every role follows a proven 3-phase learning structure:

#### Phase 1: Foundation (60-80 hours)
**Goal:** Build solid fundamentals

- Core concepts and theory
- Essential tools setup
- First hands-on projects
- Basic best practices
- Foundation assessment

**Time:** 8-12 weeks (5-10 hours/week)

**Success Criteria:**
- Understand core concepts
- Complete foundational projects
- Pass foundation assessment (75%+)
- Ready for intermediate level

#### Phase 2: Intermediate (80-120 hours)
**Goal:** Develop professional competency

- Advanced patterns and practices
- Real-world applications
- Best practices and optimization
- Team collaboration
- Building portfolio projects

**Time:** 12-16 weeks (8-12 hours/week)

**Success Criteria:**
- Build 2-3 substantial projects
- Demonstrate best practices
- Understand architecture patterns
- Ready for professional roles

#### Phase 3: Advanced (100-150 hours)
**Goal:** Achieve expertise and leadership

- Specialized techniques
- Large-scale systems
- Architecture decisions
- Mentoring others
- Innovation and thought leadership

**Time:** 14-20 weeks (10-15 hours/week)

**Success Criteria:**
- Master advanced concepts
- Architect complex systems
- Lead teams or projects
- Ready for senior roles

### Example: React Developer Path

**Foundation Phase**
- Week 1-2: JavaScript ES6+ fundamentals
- Week 3-4: React fundamentals and JSX
- Week 5-6: Component lifecycle and hooks
- Week 7-8: State management basics
- Week 9-10: First React projects
- Week 11-12: Assessment and next phase prep

**Intermediate Phase**
- Week 1-2: Advanced React patterns
- Week 3-4: State management (Redux/Zustand)
- Week 5-6: React Router and navigation
- Week 7-8: API integration
- Week 9-10: Testing (Jest, React Testing Library)
- Week 11-12: Performance optimization
- Week 13-14: Build portfolio project
- Week 15-16: Advanced patterns and review

**Advanced Phase**
- Week 1-2: Server-side rendering (Next.js)
- Week 3-4: Static site generation
- Week 5-6: Advanced TypeScript
- Week 7-8: Design systems
- Week 9-10: Web3 integration
- Week 11-12: Performance at scale
- Week 13-14: Architecture decisions
- Week 15-16: Mentoring and leadership
- Week 17-20: Capstone project

## â“ FAQ

### Learning
**Q: How long does each learning path take?**
A: Each path has 3 phases: Foundation (60-80h), Intermediate (80-120h), Advanced (100-150h). Total: 250-450 hours for full mastery.

**Q: Can I learn multiple roles simultaneously?**
A: Yes! You can have multiple paths active. We recommend focusing on 1-2 paths at a time.

**Q: What if I'm a complete beginner?**
A: Each Foundation phase is designed for beginners. Start with `/assess` to get recommendations.

### Career
**Q: Is the salary data accurate?**
A: Salary ranges are based on 2024 market data (US). Varies by location and experience.

**Q: How do I know which role to choose?**
A: Use `/assess` for personalized recommendations based on your skills.

**Q: Can I switch roles mid-path?**
A: Yes! Use `/path` to create a new path. Your progress is saved.

### Technical
**Q: Do I need internet connection?**
A: Initial installation requires internet. Learning content works offline.

**Q: Can I export my progress?**
A: Yes! Use `/path --export` to save your learning data.

**Q: Is my data private?**
A: All data is encrypted and stored securely. We never share without consent.
