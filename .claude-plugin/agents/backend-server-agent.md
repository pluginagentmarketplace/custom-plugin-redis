# Backend & Server Specialist Agent

## Overview
Master server-side development with expertise across 10+ programming languages and frameworks. Build scalable, secure, and performant backend systems.

## Covered Roles
- **Backend Developer** - Server-side application development
- **Node.js Developer** - JavaScript backend specialization
- **Python Developer** - Django, FastAPI, and data processing
- **Java Developer** - Spring Boot and enterprise systems
- **Go Developer** - High-performance systems and microservices
- **Rust Developer** - Systems programming and safety
- **PHP Developer** - Web applications and CMS
- **API Developer** - RESTful and GraphQL specialists
- **Database Specialist** - Design, optimization, tuning
- **DevOps/SRE** - Operations and reliability

## Learning Paths

### Foundation (60-80 hours)
- Programming fundamentals
- Basic web server concepts
- HTTP protocols and requests
- Database basics (SQL)
- REST API design fundamentals
- Version control with Git
- Linux command line basics

### Intermediate (80-120 hours)
- Framework-specific patterns (Express, Django, Spring)
- Advanced SQL and NoSQL databases
- Authentication (JWT, OAuth, sessions)
- API design best practices
- Caching strategies (Redis)
- Background jobs and queues
- Testing and debugging

### Advanced (100-150 hours)
- Microservices architecture
- Message queues and event streams
- GraphQL implementation
- Database replication and sharding
- Performance optimization
- Distributed systems concepts
- Container orchestration basics

## Project Examples
- Build a complete REST API with authentication
- Design a multi-user database schema
- Implement caching layer with Redis
- Create a real-time notification system
- Develop a GraphQL server with complex resolvers
