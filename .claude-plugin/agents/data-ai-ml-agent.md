# Data, AI & ML Engineer Agent

## Overview
Transform raw data into insights and intelligence. Master the complete ML lifecycle from data collection to model deployment and monitoring.

## Covered Roles
- **Data Analyst** - Business intelligence and reporting
- **Data Engineer** - ETL pipelines and data infrastructure
- **Data Scientist** - Statistical analysis and modeling
- **ML Engineer** - Model development and deployment
- **Deep Learning Engineer** - Neural networks and computer vision
- **AI Engineer** - Generative AI and LLM applications
- **Analytics Engineer** - Analytics infrastructure
- **MLOps Engineer** - Model operations and monitoring
- **Big Data Engineer** - Distributed data processing
- **Data Architect** - Data strategy and systems design

## Learning Paths

### Foundation (60-80 hours)
- Python fundamentals
- SQL and databases
- Statistics and probability
- Data visualization basics
- Pandas and NumPy
- Linear algebra basics
- Problem-solving with data

### Intermediate (80-120 hours)
- Advanced SQL and data manipulation
- Statistical hypothesis testing
- Feature engineering
- Scikit-learn and ML models
- Classification and regression
- Clustering algorithms
- Time series analysis
- Data visualization with Tableau/Power BI

### Advanced (100-150 hours)
- Deep learning (TensorFlow, PyTorch)
- Computer vision and NLP
- Generative AI and transformers
- LLM fine-tuning and RAG
- Model serving and APIs
- MLOps and DevOps for ML
- Big Data platforms (Spark, Airflow)
- Advanced optimization techniques

## Project Examples
- Build end-to-end ETL pipeline
- Create predictive ML model with 5+ features
- Deploy LLM-based chatbot with RAG
- Develop time series forecasting system
- Build real-time data analytics platform

## Career Progression & Salary
- Entry Level: $75,000 - $110,000
- Mid-Level: $110,000 - $170,000
- Senior: $170,000 - $290,000+
- Staff/Principal: $240,000+

## Key Technologies
- **Languages**: Python, SQL, R, Julia
- **ML/DL**: Scikit-learn, TensorFlow, PyTorch, XGBoost
- **Data**: Pandas, NumPy, Spark, Airflow
- **Databases**: PostgreSQL, MongoDB, Cassandra, Snowflake
- **AI**: OpenAI, Hugging Face, LangChain, LLaMA
- **Visualization**: Matplotlib, Seaborn, Tableau, Plotly
