# DevOps & Cloud Architect Agent

## Overview
Design, deploy, and manage scalable cloud infrastructure. Master containerization, orchestration, and automation for modern DevOps practices.

## Covered Roles
- **DevOps Engineer** - CI/CD and infrastructure
- **Cloud Architect** - AWS, GCP, or Azure specialist
- **SRE (Site Reliability Engineer)** - Reliability and automation
- **Infrastructure Engineer** - IaC and systems management
- **Platform Engineer** - Platform and services design
- **Kubernetes Administrator** - Container orchestration
- **Cloud Security Engineer** - Security and compliance
- **Database Administrator** - Database management and optimization
- **Terraform/IaC Specialist** - Infrastructure automation
- **DevOps Tech Lead** - Team leadership and strategy

## Learning Paths

### Foundation (60-80 hours)
- Linux fundamentals and CLI
- Git and version control workflows
- Basic Docker concepts
- Cloud platform basics (AWS/GCP)
- Introduction to CI/CD
- Basic networking concepts
- Shell scripting

### Intermediate (80-120 hours)
- Advanced Docker and container best practices
- Kubernetes basics and deployments
- AWS EC2, S3, RDS, VPC
- CI/CD with Jenkins, GitHub Actions
- Infrastructure as Code fundamentals
- Database replication and backups
- Monitoring with Prometheus, ELK
- SSL/TLS and encryption

### Advanced (100-150 hours)
- Kubernetes advanced (Helm, operators, networking)
- Terraform advanced patterns
- Multi-cloud and hybrid deployments
- Database clustering and scaling
- Disaster recovery and failover
- Cost optimization strategies
- Security hardening
- Large-scale distributed systems

## Project Examples
- Set up a complete CI/CD pipeline
- Deploy Kubernetes cluster with 10+ services
- Implement infrastructure as code for company systems
- Design multi-region failover architecture
- Implement comprehensive monitoring solution

## Career Progression & Salary
- Entry Level: $80,000 - $120,000
- Mid-Level: $120,000 - $180,000
- Senior: $180,000 - $300,000+
- Staff/Principal: $250,000+

## Key Technologies
- **Container**: Docker, Docker Compose, containerd
- **Orchestration**: Kubernetes, Docker Swarm
- **Cloud**: AWS, GCP, Azure, DigitalOcean
- **IaC**: Terraform, CloudFormation, Ansible
- **CI/CD**: Jenkins, GitHub Actions, GitLab CI
- **Monitoring**: Prometheus, Grafana, ELK, DataDog
