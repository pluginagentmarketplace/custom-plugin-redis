# Frontend & Web Specialist Agent

## Overview
Master modern web development with comprehensive coverage of 10+ specialized roles. From HTML fundamentals to advanced Web3 interfaces, this agent guides you through the complete frontend ecosystem.

## Covered Roles
- **Frontend Developer** - HTML, CSS, JavaScript core expertise
- **React Developer** - React-focused role with component design
- **Vue.js Developer** - Vue ecosystem specialization
- **Angular Developer** - Enterprise Angular applications
- **Next.js Developer** - Full-stack React with server-side rendering
- **Frontend Architect** - Large-scale systems and design
- **UI/UX Engineer** - Design implementation and user experience
- **Web Performance Engineer** - Optimization and Core Web Vitals
- **Web3 Frontend Developer** - Blockchain interfaces and dApps
- **TypeScript Developer** - Type-safe JavaScript development

## Learning Paths

### Foundation (60-80 hours)
- HTML5 semantic structure
- CSS3 fundamentals and Flexbox/Grid
- JavaScript ES6+ basics
- DOM manipulation and events
- Basic React or Vue fundamentals
- Git and version control

### Intermediate (80-120 hours)
- Advanced CSS (animations, transforms)
- TypeScript type system
- Modern state management
- Component libraries and Storybook
- Testing (unit, integration)
- Performance profiling
- SEO and accessibility

### Advanced (100-150 hours)
- Advanced patterns (HOC, render props, composition)
- Server-side rendering (SSR) with Next.js
- API integration and data fetching
- Design systems at scale
- Web Workers and Service Workers
- Web3 integration and wallets
- Advanced TypeScript patterns

## Project Examples
- Build a responsive e-commerce product page
- Create a real-time chat application with React
- Develop a design system component library
- Build a blockchain dApp interface
- Optimize performance of a large web application
