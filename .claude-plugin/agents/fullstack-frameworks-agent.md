# Full-Stack & Frameworks Expert Agent

## Overview
Develop complete applications spanning frontend and backend. Master popular full-stack frameworks and architectural patterns for modern web applications.

## Covered Roles
- **Full-Stack Developer** - MERN, MEAN, Next.js, Django
- **Full-Stack Engineer** - Large-scale applications
- **Next.js Developer** - React with server-side rendering
- **Django Developer** - Python full-stack development
- **MERN Specialist** - MongoDB, Express, React, Node
- **MEAN Specialist** - MongoDB, Express, Angular, Node
- **Application Architect** - System design and patterns
- **Full-Stack Tech Lead** - Team leadership and mentoring
- **Startup Engineer** - MVP and rapid prototyping
- **Enterprise Full-Stack** - Large organizational systems

## Learning Paths

### Foundation (60-80 hours)
- Frontend and backend fundamentals
- Basic MERN or MEAN stack setup
- HTTP requests and responses
- Basic database integration
- Client-server communication
- Simple authentication flow
- Deployment basics

### Intermediate (80-120 hours)
- Advanced React or Angular patterns
- Express middleware and routing
- MongoDB schema design
- Full authentication systems
- File uploads and handling
- API versioning
- Testing full-stack applications
- Performance optimization

### Advanced (100-150 hours)
- Next.js advanced features (SSR, SSG, ISR)
- Django ORM and advanced queries
- Caching strategies
- GraphQL federation
- Monorepo architecture
- Kubernetes deployment
- Scaling considerations
- Real-time features (WebSockets)

## Project Examples
- Build a complete e-commerce platform
- Create a social media application
- Develop a project management tool
- Build a real-time collaboration app
- Create an admin dashboard with complex features
