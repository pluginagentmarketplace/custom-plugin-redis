# Security & QA Expert Agent

## Overview
Protect applications and systems from vulnerabilities. Master security practices, penetration testing, and comprehensive quality assurance processes.

## Covered Roles
- **QA Engineer** - Manual and automated testing
- **Automation Tester** - QA automation specialist
- **Security Analyst** - Vulnerability and threat analysis
- **Penetration Tester** - Ethical hacking and assessment
- **Security Engineer** - Secure system design
- **Compliance Specialist** - Regulations and standards
- **Security Architect** - Enterprise security design
- **DevSecOps Engineer** - Security in CI/CD
- **Incident Response Specialist** - Crisis management
- **Application Security (AppSec) Engineer** - Secure development

## Learning Paths

### Foundation (60-80 hours)
- OWASP Top 10 vulnerabilities
- Basic security principles
- Testing methodologies
- SQL basics and injection concepts
- XSS and CSRF fundamentals
- Authentication methods
- QA fundamentals and testing types

### Intermediate (80-120 hours)
- Advanced web application hacking
- Burp Suite and security tools
- Secure coding practices
- Cryptography fundamentals
- Network security basics
- Advanced QA automation (Selenium, Cypress)
- API security testing
- GDPR and compliance basics

### Advanced (100-150 hours)
- Advanced penetration testing
- Zero-day exploitation research
- Red team exercises
- Security architecture design
- Compliance frameworks (HIPAA, PCI-DSS, ISO 27001)
- Security incident response
- Threat modeling
- Advanced automation frameworks

## Project Examples
- Perform penetration test on target application
- Design comprehensive QA automation suite
- Implement security scanning in CI/CD
- Conduct security code review
- Create incident response playbook

## Career Progression & Salary
- Entry Level: $70,000 - $110,000
- Mid-Level: $110,000 - $175,000
- Senior: $175,000 - $290,000+
- Staff/Principal: $240,000+

## Key Technologies
- **Security Tools**: Burp Suite, Metasploit, Wireshark, Nessus
- **QA Tools**: Selenium, Cypress, Appium, TestNG, JUnit
- **Languages**: Python, JavaScript, Bash, PowerShell
- **Frameworks**: OWASP, NIST, PCI-DSS, GDPR
- **Testing**: BDD, TDD, Gray-box testing, Fuzzing
- **DevSecOps**: SAST, DAST, SCA, IAST tools
