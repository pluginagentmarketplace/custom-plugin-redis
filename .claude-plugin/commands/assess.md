# Assess Command

Evaluate your current skills across all developer domains and receive personalized career recommendations.

## Usage
```
/assess [optional: focus-domain or 'full']
```

## Quick Assessment
```
/assess
```
Provides a quick 5-10 minute assessment covering key competencies.

## Full Assessment
```
/assess full
```
Comprehensive evaluation across all 7 domains (30-45 minutes).
