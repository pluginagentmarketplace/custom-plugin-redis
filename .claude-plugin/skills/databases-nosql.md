# NoSQL & Data Stores

## Learning Objectives
- Choose appropriate NoSQL database for use cases
- Design document-oriented schemas
- Implement caching strategies
- Build search and analytics solutions
- Handle distributed data
- Scale horizontally

## Core Topics
1. **Document Databases (MongoDB)**
   - Document structure
   - CRUD operations
   - Aggregation pipeline
   - Indexing strategies
   - Transactions

2. **Key-Value Stores (Redis)**
   - Data types (strings, hashes, lists, sets)
   - Expiration and TTL
   - Pub/Sub messaging
   - Caching patterns
   - Lua scripting

3. **Search Engines (Elasticsearch)**
   - Indexing and mapping
   - Search queries
   - Aggregations
   - Full-text search
   - Log analysis

4. **Other NoSQL**
   - Cassandra for distributed data
   - DynamoDB for serverless
   - Graph databases
   - Time-series databases

## Patterns & Best Practices
- Denormalization patterns
- Sharding strategies
- Consistency models
- Caching layers
- Change data capture

## Projects
- Real-time caching layer
- Full-text search implementation
- Log aggregation system
- Leaderboard with Redis
- Document database design

## Assessment
- Implement caching strategy
- Build search feature
- Design document schema
