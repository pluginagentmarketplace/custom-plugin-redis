# SQL & Relational Databases

## Learning Objectives
- Design efficient database schemas
- Write optimized SQL queries
- Implement relationships and constraints
- Manage transactions and ACID compliance
- Monitor and optimize database performance
- Handle backup and recovery

## Core Topics
1. **SQL Fundamentals**
   - SELECT, INSERT, UPDATE, DELETE
   - WHERE, JOIN, GROUP BY, ORDER BY
   - Aggregation functions
   - Subqueries and CTEs
   - Window functions

2. **Schema Design**
   - Normalization (1NF-3NF)
   - Primary and foreign keys
   - Indexes and performance
   - Constraints and triggers
   - Entity-relationship diagrams

3. **Advanced SQL**
   - Complex joins and set operations
   - Window functions and analytics
   - JSON operations
   - Full-text search
   - Recursive queries

4. **Database Administration**
   - User roles and permissions
   - Backup and restore
   - Replication
   - Performance monitoring
   - Query optimization

## Tools & Technologies
- PostgreSQL advanced features
- MySQL and MariaDB
- SQLite for embedded use
- Query optimization
- Monitoring tools

## Projects
- E-commerce database schema
- Analytics database design
- Time-series data storage
- Multi-tenant database
- Query optimization challenge

## Assessment
- Design normalized schema
- Write complex queries
- Optimize slow queries
- Implement backup strategy
