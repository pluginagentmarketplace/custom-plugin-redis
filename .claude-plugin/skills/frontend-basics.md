# Frontend Fundamentals

## Learning Objectives
- Write semantic HTML5 with accessibility in mind
- Style websites with CSS3 including Flexbox and Grid
- Write JavaScript ES6+ with async/await and Promises
- Manipulate the DOM and handle events
- Create responsive designs for multiple devices
- Understand browser DevTools for debugging

## Core Topics
1. **HTML5 & Semantic Markup**
   - Semantic HTML elements
   - Forms and validation
   - Accessibility (ARIA)
   - Meta tags and SEO

2. **CSS3 Fundamentals**
   - Selectors and specificity
   - Box model and positioning
   - Flexbox layout
   - CSS Grid
   - Responsive design with media queries
   - Animations and transitions

3. **JavaScript Basics**
   - Variables, types, and operators
   - Functions and scope
   - Objects and arrays
   - Prototypes and this binding
   - Closures and higher-order functions
   - Callbacks and Promises
   - Async/await syntax

4. **DOM & Events**
   - Selecting elements
   - Manipulating content and attributes
   - Event listeners and handlers
   - Event delegation
   - DOM traversal

## Projects
- Personal portfolio website
- Responsive landing page
- Interactive to-do list application
- Form with validation
- Image carousel component

## Assessment
- Quiz on HTML semantics
- CSS layout challenges
- JavaScript debugging exercises
- Build a responsive website from mockup
