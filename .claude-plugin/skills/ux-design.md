# UX/UI Design & Design Systems

## Learning Objectives
- Understand UX principles and methodology
- Design user-centered interfaces
- Conduct user research
- Build design systems
- Implement accessibility standards
- Measure design impact

## Core Topics
1. **UX Fundamentals**
   - User research methods
   - Personas and user flows
   - Wireframing and prototyping
   - Usability testing
   - User journey mapping

2. **UI Design Principles**
   - Visual hierarchy
   - Color theory
   - Typography
   - Layout and spacing
   - Visual consistency

3. **Design Systems**
   - Component libraries
   - Design tokens
   - Brand guidelines
   - Pattern documentation
   - Maintenance and evolution

4. **Accessibility (WCAG)**
   - Color contrast
   - Keyboard navigation
   - Screen reader support
   - Semantic HTML
   - ARIA attributes

5. **Design Tools & Process**
   - Figma
   - Adobe XD
   - Prototyping tools
   - Design handoff
   - Developer collaboration

## Tools & Platforms
- Figma for design
- Adobe Creative Suite
- Sketch
- Storybook
- Design system frameworks

## Projects
- Complete UI redesign
- Design system creation
- Accessibility audit
- Usability testing session
- Design specifications

## Assessment
- User research findings
- Design system documentation
- Prototype evaluation
- Accessibility assessment
- Design impact metrics
