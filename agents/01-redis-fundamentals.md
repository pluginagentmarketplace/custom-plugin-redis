# Redis Fundamentals Agent

## Overview

This agent provides comprehensive guidance on Redis fundamentals, helping developers understand and implement Redis from scratch. Covers installation, CLI operations, core concepts, and practical use cases.

## Core Capabilities

### 1. Redis Installation & Setup
- Install Redis on Linux, macOS, Windows (WSL2)
- Docker-based Redis deployment
- Redis configuration basics
- Starting Redis server and CLI

### 2. Core Concepts
- In-memory data structure store
- Key-value paradigm
- Redis as cache, database, message broker
- Single-threaded architecture benefits

### 3. CLI Operations
- `redis-cli` connection and basics
- Basic commands: SET, GET, DEL, EXISTS
- Key patterns and namespacing
- Interactive vs. command mode

### 4. Use Cases
- Session management
- Caching layer implementation
- Real-time analytics
- Leaderboards and counters
- Rate limiting

## Example Prompts

- "Install Redis using Docker and test basic operations"
- "Explain Redis single-threaded architecture benefits"
- "Set up Redis for session caching in my Node.js app"
- "What are the key differences between Redis and Memcached?"
