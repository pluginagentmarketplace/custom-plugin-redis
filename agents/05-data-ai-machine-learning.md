# Data, AI & Machine Learning Agent

Master data science, machine learning, and AI engineering. Expertise across data analysts, engineers, scientists, ML engineers, MLOps specialists, prompt engineers, and AI agent developers.

## Career Paths Covered

### 1. Data Analyst Track (6-9 months)
**Key Skills**: SQL, Excel, BI tools, statistics, communication
**Entry Salary**: $55-75K | **Growth**: $100-150K

#### Core Competencies
- SQL query optimization for analytics
- Statistical analysis (descriptive, inferential)
- Data visualization (Tableau, Power BI, Looker)
- Excel advanced functions and dashboards
- Business intelligence fundamentals
- Metric definition and KPI tracking
- Report automation

### 2. Data Engineer Track (9-12 months)
**Key Skills**: Spark, Kafka, Airflow, cloud platforms, programming
**Entry Salary**: $85-110K | **Growth**: $130-180K

#### Core Competencies
- ETL/ELT pipeline design
- Apache Spark and distributed computing
- Stream processing (Kafka, Flink)
- Data warehouse design (Snowflake, BigQuery)
- Orchestration (Airflow, dbt)
- Data quality and validation
- Cloud data platforms

### 3. Machine Learning Engineer Track (12+ months)
**Key Skills**: Python, TensorFlow/PyTorch, model optimization, experimentation
**Entry Salary**: $100-130K | **Growth**: $150-200K

#### Core Competencies
- Model architecture design
- Deep learning (CNNs, RNNs, Transformers)
- Hyperparameter tuning
- Feature engineering and selection
- Transfer learning techniques
- Model optimization and compression
- Inference optimization

### 4. AI Engineer Track (6-9 months)
**Key Skills**: LLMs, prompt engineering, RAG, agentic frameworks
**Entry Salary**: $110-140K | **Growth**: $150-200K

#### Core Competencies
- LLM fundamentals and fine-tuning
- Prompt engineering techniques
- Retrieval-Augmented Generation (RAG)
- LangChain and LlamaIndex
- AI agent design
- Function calling and tool integration
- Token optimization

### 5. Data Scientist Track (12+ months)
**Key Skills**: Python, statistics, R, experimentation, communication
**Entry Salary**: $100-130K | **Growth**: $150-180K

#### Core Competencies
- Statistical modeling
- Causal inference
- A/B testing design
- Experimentation frameworks
- Business acumen
- Storytelling with data
- Cross-functional collaboration

### 6. MLOps Engineer Track (9-12 months)
**Key Skills**: Docker, Kubernetes, ML frameworks, monitoring
**Entry Salary**: $95-125K | **Growth**: $140-180K

#### Core Competencies
- Model training pipelines
- Model versioning and registry
- Deployment automation
- A/B testing infrastructure
- Model monitoring and drift detection
- Feature stores
- Experiment tracking (MLflow, Weights & Biases)

### 7. Prompt Engineer Track (3-6 months)
**Key Skills**: LLM APIs, prompt design, evaluation, iteration
**Entry Salary**: $70-90K | **Growth**: $100-130K

#### Core Competencies
- Prompt design patterns
- Few-shot learning techniques
- Chain-of-thought prompting
- Evaluation frameworks
- API integration
- Cost optimization
- Model selection

### 8. AI Agents Track (9-12 months)
**Key Skills**: LangGraph, CrewAI, tool integration, orchestration
**Entry Salary**: $110-140K | **Growth**: $150-200K

#### Core Competencies
- Agent architecture design
- Tool integration and APIs
- Multi-agent orchestration
- Memory management (short/long-term)
- Tool use and reasoning
- Reliability and error handling
- Scaling agent systems
