# /assess - Knowledge Assessment & Testing

Evaluate your knowledge across different technologies and roles. Get personalized feedback and recommendations.

## Usage

```
/assess
/assess {topic}
/assess {agent-id}
```

## Assessment Types

### 1. Topic-Specific Assessments
Test knowledge on specific technologies:

```
/assess react
/assess typescript
/assess postgresql
/assess kubernetes
/assess machine-learning
```

**What You'll Get**:
- 10-20 questions per topic
- Multiple choice and hands-on coding challenges
- Instant feedback on each answer
- Score breakdown by concept
- Recommended learning resources

### 2. Agent-Based Assessments
Evaluate readiness for different roles:

```
/assess frontend-web-agent
/assess backend-database-agent
/assess data-ai-ml-agent
```

**What You'll Get**:
- Comprehensive skill evaluation
- Role-specific competency assessment
- Career readiness score
- Gap analysis with recommendations
- Suggested learning path

### 3. Full Learning Path Assessment
Evaluate progress on complete roadmaps:

```
/assess react-developer
/assess backend-fundamentals
/assess full-stack-javascript
```

**What You'll Get**:
- 50-100 questions covering full path
- Progress by phase
- Detailed competency report
- Estimated completion time
- Next milestone recommendations

7. **Share results** - Get feedback from agents
