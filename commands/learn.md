# /learn - Interactive Learning Paths

Welcome to the Developer Roadmap Interactive Learning System! This command helps you start your personalized learning journey.

## Usage

```
/learn
```

## What It Does

The `/learn` command presents you with:

1. **Role Selection** - Choose from 70+ different developer roles
2. **Learning Paths** - Get customized progression paths based on your experience level
3. **Time Estimates** - Understand how long each path typically takes
4. **Next Steps** - Get directed to the right agent and resources

## Experience Levels

- **Beginner**: No professional experience, starting fresh
- **Intermediate**: 1-3 years in related field, looking to specialize
- **Advanced**: 3+ years, transitioning to new specialization
- **Expert**: Deep expertise, looking to lead or pivot

## Interactive Features

The `/learn` command guides you through:

1. **Skill Assessment** - Self-evaluate your current skills
2. **Goal Setting** - Define what you want to achieve
3. **Personalized Roadmap** - Get a tailored learning path
4. **Resource Recommendations** - Curated learning materials
5. **Progress Tracking** - Monitor your advancement
6. **Milestone Celebrations** - Get recognized for achievements

## Learning Path Structure

Each learning path includes:

- **Phases** (2-4 progressive stages)
- **Core Topics** (20-50 topics per phase)
- **Subtopics** (100-200 subtopics total)
- **Best Practices** (10-20 per specialization)
- **Hands-On Projects** (3-5 projects)
- **Assessment Quizzes** (per topic)
- **Time Estimates** (in hours)
- **Difficulty Levels** (Beginner ‚Üí Intermediate ‚Üí Advanced)

## Tips for Success

1. **Start with foundations** - Don't skip basics even if you think you know them
2. **Do projects** - Build real things to cement knowledge
3. **Take assessments** - Evaluate your understanding
4. **Review regularly** - Revisit past topics monthly
5. **Join community** - Connect with others on the same path
6. **Ask questions** - Use `/browse-agent` to get specific help
7. **Celebrate wins** - Acknowledge progress and milestones
