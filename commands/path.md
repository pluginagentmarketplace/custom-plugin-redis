# /path - Create Your Learning Path

Design your personalized learning journey based on goals.

## Usage
```
/path
/path frontend-react
/path backend-python
/path devops-kubernetes
```

## Path Creation Process

1. **Choose Your Goal**
   - Select target role from 65+ available
   - Specify experience level (Beginner/Intermediate/Advanced)

2. **Get Custom Roadmap**
   - Foundation phase with prerequisites
   - Core skills and competencies
   - Advanced topics and specializations
   - Hands-on projects

3. **Timeline & Resources**
   - Estimated duration (weeks/months)
   - Time commitment needed
   - Learning resources
   - Tools and technologies

4. **Track Progress**
   - Milestone-based structure
   - Skill checkpoints
   - Assessment points
   - Project portfolio

## Example Paths

### Frontend React Developer
- Weeks 1-4: HTML, CSS, JavaScript
- Weeks 5-8: React fundamentals
- Weeks 9-12: Advanced React patterns
- Weeks 13-16: Projects and portfolio

### Backend Python Developer
- Weeks 1-4: Python fundamentals
- Weeks 5-8: Django framework
- Weeks 9-12: Databases and APIs
- Weeks 13-16: Deployment and DevOps basics

### DevOps Engineer
- Weeks 1-4: Linux, Docker
- Weeks 5-8: Kubernetes basics
- Weeks 9-12: AWS or cloud platform
- Weeks 13-16: CI/CD and production systems
