# /redis-learn - Master Redis Progressively

Learn Redis from fundamentals to advanced with structured paths.

## Usage

```
/redis-learn
/redis-learn beginner
/redis-learn intermediate
/redis-learn advanced
```

## Learning Paths

### Beginner Path (20-30 hours)

**Foundation**: Start here if you're new to Redis

1. **What is Redis?** (2 hours)
   - In-memory data store
   - Key-value structure
   - Use cases
   - When to use Redis

2. **Installation & Setup** (2 hours)
   - Install Redis
   - Run redis-server
   - Use redis-cli
   - Configuration basics

3. **Data Structures Basics** (6 hours)
   - Strings: GET, SET, INCR
   - Lists: LPUSH, RPOP, BLPOP
   - Sets: SADD, SMEMBERS, SINTER
   - Hashes: HSET, HGET, HGETALL
   - Sorted Sets: ZADD, ZRANGE

4. **Expiration & TTL** (3 hours)
   - EXPIRE, SETEX
   - TTL management
   - Cleanup strategies

5. **Simple Caching** (4 hours)
   - Cache-aside pattern
   - Cache invalidation
   - Simple example: Web page cache

6. **Hands-On Project** (8 hours)
   - Build simple cache layer
   - Counter application
   - Session storage
   - Quiz with scoring

### Intermediate Path (30-40 hours)

**Production Ready**: Build with Redis

1. **Advanced Data Structures** (8 hours)
   - Bitmaps: SETBIT, BITCOUNT
   - HyperLogLog: PFADD, PFCOUNT
   - Streams: XADD, XREAD
   - Deep dives on all structures

2. **Persistence** (4 hours)
   - RDB snapshots
   - AOF logs
   - Hybrid setup
   - Disaster recovery

3. **Transactions & Lua** (6 hours)
   - MULTI/EXEC
   - WATCH for optimistic locking
   - Lua scripting for atomic ops
   - Complex operations

4. **Pub/Sub Messaging** (4 hours)
   - Publish/Subscribe
   - Pattern matching
   - Real-time applications
   - Notification systems

5. **Replication** (4 hours)
   - Master-Slave setup
   - Monitoring replication
   - Replication lag
   - Multi-replica scenarios

6. **Hands-On Project** (8 hours)
   - Build real-time chat
   - Queue system
   - Session manager
   - Leaderboard

### Advanced Path (40-50 hours)

**Expert**: Production systems at scale

1. **Cluster Setup** (6 hours)
   - Cluster architecture
   - Node configuration
   - Slot management
   - Scaling horizontally

2. **Sentinel & HA** (5 hours)
   - Automatic failover
   - Sentinel monitoring
   - Multi-sentinel setup
   - Consensus algorithms

3. **Performance Tuning** (6 hours)
   - Pipelining
   - Connection pooling
   - Memory optimization
   - Benchmarking

4. **Production Patterns** (6 hours)
   - Complex caching
   - Distributed locks
   - Rate limiting
   - Session management
   - Queues with retries

5. **Monitoring & Operations** (5 hours)
   - Slowlog analysis
   - INFO command
   - Custom metrics
   - Alerting setup

6. **Capstone Project** (12 hours)
   - Build scalable system
   - HA with Sentinel
   - Multiple patterns
   - Monitoring dashboard
